The aim of this course is to familiarize students with several supervised learning techniques and the Scikit-learn python library. This library is designed for machine learning. Here, we'll be focusing on classification problems and algorithms/techniques.
In particular, we'll be using decision trees, SVM, bagging, random forests and boosting techniques...
